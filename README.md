# Agro Shop — Site estático (demo)

Descrição:
Site frontend estático de uma loja virtual para agronegócios chamado "Agro Shop". Implementado com HTML/CSS/JS simples. Possui:
- Catálogo de produtos;
- Modal de detalhe do produto;
- Carrinho com persistência via localStorage;
- Checkout simulado.

Como usar:
1. Salve os arquivos em uma pasta com a estrutura:
   - index.html
   - css/styles.css
   - js/shop.js
   - README.md
2. Abra `index.html` em qualquer navegador moderno.
3. Teste: adicionar produtos, abrir carrinho, finalizar compra (simulado).
